# Password Generator (Python)

This is a secure password generator built using Python.

## Features
- User-defined password length
- Uses cryptographically secure secrets module
- Includes letters, digits, and symbols

## Technologies Used
- Python
- secrets
- string

## How to Run
1. Install Python
2. Run the script:
   python password_generator.py
